# Banco-de-Dados-1
# Acesso ao Sistema de Banco de Dados - Supermercado Castelus

1. No terminal, dê o comando: git clone https://github.com/catamartins/SGBD-Supermercado.git
   
3. Rode o script.txt para criar as tabelas e populá-las
   
4. Dê o comando: python -m venv venv -- criação de ambiente virtual
   
5. Dê o comando: .\venv\Scripts\activate -- ativa ambiente
   
6. Dê o comando: pip install -r requirements.txt -- instalação das dependencias
   
7. Dẽ o comando: python main.py
   
8. Pesquise no seu navegador: http://127.0.0.1:8000/

9. Faça login a partir das visões:
- Diretoria geral:
    Usuário: diretor |
    Senha: 1234
- Financeiro:
    Usuário: financeiro |
    Senha: 1234
- Frente de Caixa:
    Usuário: caixa |
    Senha: 1234
